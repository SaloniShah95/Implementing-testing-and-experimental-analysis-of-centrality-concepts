To run code:
1. Put the layers you want to run the code on in the folder Heuristics.
2. Go to  the file layerComposerConfiguration and change the line #5 and #7 with the new names of layers.
3. Run command: python betweeness_heuristic.py <layer1> <layer2>
   For e.g. python betweeness_heuristic.py American Delta
